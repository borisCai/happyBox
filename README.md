# 🧮 小学二年级数学速算比赛

一个专为小学二年级学生设计的数学速算练习单页应用，帮助提升计算能力和反应速度。

## ✨ 项目特色

- 📱 **纯HTML单页应用**：无需后端服务，打开即用
- 📱 **多端自适应**：完美支持电脑端、平板端、手机端
- 🎯 **两种比赛模式**：倒计时比赛和常规比赛
- 📊 **成绩自动记录**：本地存储，查看历史成绩和详细答题记录
- 🎨 **美观的UI设计**：现代化界面，流畅的交互动画

## 🚀 功能特性

### 比赛模式

#### ⏱️ 倒计时比赛模式
- **时长**：3分钟（180秒）
- **目标**：在时间内尽可能完成更多题目
- **适合**：训练计算速度和反应能力

#### 🎯 常规比赛模式
- **时长**：5分钟（300秒）
- **题目数量**：最多70道题
- **特点**：题目完成后可提前结束
- **适合**：追求准确率与速度的平衡

### 题目类型

1. **100以内加法**：确保结果不超过100
2. **100以内减法**：确保结果非负
3. **九九乘法表**：1-9的乘法运算
4. **除法口诀**：九九乘法表对应的除法，确保能整除
5. **混合运算题**（必须包含5道）：
   
   **带括号的混合运算**：
   - 加法乘法：`(a + b) × c`
   - 加法除法：`(a + b) ÷ c`
   - 减法乘法：`(a - b) × c`
   - 减法除法：`(a - b) ÷ c`
   
   **不带括号的混合运算**（先乘除后加减）：
   - 先乘后加：`a + b × c`
   - 先除后加：`a + b ÷ c`
   - 先乘后减：`a - b × c`
   - 先除后减：`a - b ÷ c`

### 核心功能

- ✅ **逐题解答**：每道题提交后自动进入下一题
- ⌨️ **自动唤醒键盘**：手机端自动弹出数字键盘
- ⏰ **实时倒计时**：显示剩余时间，最后30/60秒红色警告
- 📊 **实时统计**：显示题目进度、正确题数、已答题数
- 💾 **成绩记录**：自动保存每次比赛成绩
- 📝 **详细记录**：查看每道题的答题情况、正确答案和解析

## 🎮 使用方法

1. **打开页面**：直接用浏览器打开 `index.html` 文件
2. **选择模式**：在首页选择"倒计时比赛"或"常规比赛"
3. **开始答题**：
   - 输入答案后点击"提交答案"按钮
   - 或直接按回车键提交
   - 提交后自动进入下一题
4. **查看成绩**：在"我的成绩"标签页查看历史记录和详细答题情况

## 📱 界面说明

### 首页
- 显示两种比赛模式卡片
- 点击卡片即可开始对应模式的比赛

### 比赛规则
- 详细说明两种比赛模式的规则
- 说明题目类型和计分规则

### 我的成绩
- 以卡片形式展示所有历史成绩
- 显示完成时间、完成题数、正确题数、正确率
- 点击"查看详细记录"可展开查看每道题的：
  - 题目内容
  - 你的答案 vs 正确答案
  - 题目类型
  - 答题状态（正确/错误）
  - 解析说明（混合运算题）

### 游戏界面
- **顶部信息栏**：显示题目进度、正确题数、已答题数、倒计时
- **题目区域**：大字体显示当前题目
- **输入区域**：数字输入框和提交按钮

## 💾 数据存储

- 使用浏览器 `localStorage` 本地存储
- 最多保存50条成绩记录
- 自动保存，无需手动操作
- 数据保存在浏览器本地，清除浏览器数据会丢失

## 🎯 适用对象

- 小学二年级学生
- 需要提升计算能力的儿童
- 家长辅助孩子练习数学
- 教师作为课堂练习工具

## 📋 技术特点

- **纯前端实现**：HTML + CSS + JavaScript
- **无需依赖**：不依赖任何外部库或框架
- **响应式设计**：CSS媒体查询适配多端
- **本地存储**：使用 localStorage API
- **现代浏览器支持**：支持所有现代浏览器

## ⚠️ 注意事项

1. 建议使用现代浏览器（Chrome、Firefox、Safari、Edge等）
2. 手机端建议横屏使用以获得更好的体验
3. 清除浏览器数据会删除所有保存的成绩记录
4. 题目为随机生成，每次比赛题目不同
